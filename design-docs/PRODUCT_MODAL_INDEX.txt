================================================================================
CUSTOMIZATION POINTS
================================================================================

COLORS
  Primary action (Add to Cart)    → Orange (#ff6b35)
  Veg indicator                   → Green
  Non-veg indicator               → Red
  Required badge                  → Orange
  Selected state                  → Orange border + light background
  Hover states                    → Muted background

CONTENT
  Quick request suggestions       → Edit QUICK_REQUESTS array
  Size options                    → Configure product.availableSizes
  Addon categories                → Add category to addon objects
  Character limits                → Change maxLength on textarea

BEHAVIOR
  Max addon quantity              → Set maxQuantity per addon
  Special request limit           → Change 200 to preferred limit
  Quantity range                  → Adjust min/max in counter
  Required fields                 → Mark with isRequired prop
