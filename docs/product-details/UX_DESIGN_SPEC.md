## Design Philosophy

### Core Principles

1. **Speed to Order**: Food delivery users want to order quickly. Every interaction should reduce friction and decision fatigue.

2. **Appetizing Presentation**: The product image should be the hero. Make it impossible not to crave the food.

3. **Clear Value Proposition**: Price should always be visible, with clear breakdowns of what adds cost.

4. **Premium Feel**: Subtle animations and polished interactions create perceived value, justifying premium pricing.

5. **Thumb-Friendly Mobile**: 80% of food orders happen on mobile. Design for one-handed use.

### UX Success Metrics

| Metric | Target |
|--------|--------|
| Time to Add | < 8 seconds (simple item) |
| Selection Clarity | Zero user confusion on variants |
| Price Visibility | Always visible, never scrolled away |
| Cart Completion | > 85% once modal opened |

### Interaction Patterns

**Open Animation:**
- Backdrop fades in (200ms)
- Modal scales from 95% to 100% with opacity 0 to 1
- Spring animation: damping 25, stiffness 300
- Stagger content sections (50ms delay each)

**Close Animation:**
- Content fades first (100ms)
- Modal scales to 95% with fade (150ms)
- Backdrop fades out (200ms)

**Scroll Behavior:**
- Internal scroll within modal body
- Sticky footer remains visible
- Header can scroll away (image provides context)
- Scrollbar styled to match theme
