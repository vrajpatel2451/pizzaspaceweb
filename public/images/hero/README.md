# Hero Slider Images

This directory should contain the hero slider background images.

## Required Images

Place the following images in this directory:

- `slide1.jpg` - New Margherita Supreme pizza (recommended size: 1920x1080px)
- `slide2.jpg` - Free Delivery Weekend promotion (recommended size: 1920x1080px)
- `slide3.jpg` - Family Meal Deal (recommended size: 1920x1080px)

## Image Guidelines

- **Format**: JPG or WebP for best performance
- **Dimensions**: Minimum 1920x1080px (16:9 aspect ratio)
- **File Size**: Optimize images to be under 200KB each
- **Content**: Ensure text is readable when dark overlay is applied
- **Focus**: Keep important content in the center as it will be visible on all screen sizes

## Temporary Solution

Until you add your own images, you can use placeholder images from:
- Unsplash (https://unsplash.com/s/photos/pizza)
- Pexels (https://www.pexels.com/search/pizza/)

Download high-quality pizza images and rename them to match the required filenames above.
